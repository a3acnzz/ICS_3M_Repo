Instructions for completing the TTS Professional Services eRFE (Electronic Request for Expenditure -eRFE) form.

  • This document is used to obtain approvals and then prepare an order to be sent to 3M providers or them to supply IT related goods and services.
  • The eRFE document data is used to process orders to various 3M suppliers for the detailed items listed on the attached documents.
  • An RFE number will be assigned in AMAPS once all approvals have been completed. The RFE number will be used to track all activity regarding the order and subsequent costs that accumulate as a result of the order.
  • * = Required field.
Section Field Name Information to Enter
1 Origin Date eRFE system will enter today’s date. No entry necessary.
2 PS Request Number eRFE system will assign a unique number for reference. It is not a valid order number.
3 Project Title* Enter in a brief description of Services.
4 Start and Completion date* Start date and completion date for this request.
5 Site Contact Name* Click in the box and start typing for the name needed, the format is Last Name<space>First Name or Last Name alone (e.g. The name needs to be selected is John Smith, then enter Smith John / Smith).

Note: Partial names are also acceptable (e.g. Smi Jo / Smi) and it is case insensitive (e.g. smiT / Smit ). For wildcard searching, "%" can be added to the name as required (e.g. Smit% Joh%, %vid %lay, %poulo%). If no "%" is added it will be considered as %name%.

During foucus out, if it has a single result, it will be auto-populated else the popup with the search results will be shown. Select the contact name from the list. The site contact will receive all correspondence regarding this order.
6 Project Coordinator Click in the box and start typing for the name needed, the format is Last Name<space>First Name or Last Name alone (e.g. The name needs to be selected is John Smith, then enter Smith John / Smith).

Note: Partial names are also acceptable (e.g. Smi Jo / Smi) and it is case insensitive (e.g. smiT / Smit ). For wildcard searching, "%" can be added to the name as required (e.g. Smit% Joh%, %vid %lay, %poulo%). If no "%" is added it will be considered as %name%.

During foucus out, if it has a single result, it will be auto-populated else the popup with the search results will be shown. Select the contact name from the list. They will receive all correspondence regarding this order. If same as site contact, can be left blank.
7 Person ID Will be filled from your choice above. Please validate you have the correct name and Person ID.
8 Order From (Provider/Supplier Name)* Click in the box and start typing for the supplier name needed. During foucus out, if it has a single result, it will be auto-populated else the popup with the search results will be shown. The button prepended to the input box can be used to search for supplier name. If the supplier is not provided in the selection list, please select the very last supplier name available (ZZZZZZ new Supplier) and enter the correct supplier information in the description field. If new supplier, please contact Julie Wanek and LeAnn Windisch for supplier set up.

Note: Partial supplier names are also acceptable (e.g. Cognizant) and it is case insensitive (e.g. cognizant). For wildcard searching, "%" can be added to the name as required (e.g. Cognizant%, %Solutions, %Technology%). If no "%" is added it will be considered as %name%.
9 Type of Bid:* Fixed or Hourly
10 Worker Information If hourly provide: Resource Title (can be left blank), Estimated Hours, Rate.
11 Additional Expenses: 3M allowable expenses, example: travel costs, hotels etc…
12 Fixed Bid Amount Enter the amount of the fix bid
13 Approval Amount* Total amount being requested
14 Project Title* Short description of project/services
15 Description of all services provided* Details of the project.
16 Account* Most eRFE’s will be 7385040, if something different click and search for the account number.
17 Cost Center (Dept #)* Enter a valid cost center to receive the charges for these expenditure(s).
18 WBS Enter the WBS if needed (do not put a project name). Leave Blank if no WBS.
19 Distribution %* If one cost center is to receive all costs enter 100% as the distribution to be charged when invoices are received from the provider. If the percentage of costs to be distributed from the total committed dollars is to be split to more than one cost center, the cost must be distributed over all the cost centers on a percentage basis. Example: Two cost centers sharing equally will be 50% for each. Total of all line percentages must equal 100%. Can only have two decimals points after the period.
20 Distribution Notes: Can be left blank.
Approval may be obtained from several people as required by your management chain, however at least one of the approvers must have an approval authorization level covering the total amount to be approved.
21 Approver* Click in the box and start typing for the name needed, the format is Last Name<space>First Name or Last Name alone (e.g. The name needs to be selected is John Smith, then enter Smith John / Smith).

Note: Partial names are also acceptable (e.g. Smi Jo / Smi) and it is case insensitive (e.g. smiT / Smit ). For wildcard searching, "%" can be added to the name as required (e.g. Smit% Joh%, %vid %lay, %poulo%). If no "%" is added it will be considered as %name%.

During foucus out, if it has a single result, it will be auto-populated else the popup with the search results will be shown. Select the approver name from the list. Input all approver names as required. At least one person must have approval authority to cover the amount listed in the total approval amount.
22 Informational copy Click in the box and start typing for the name needed, the format is Last Name<space>First Name or Last Name alone (e.g. The name needs to be selected is John Smith, then enter Smith John / Smith).

Note: Partial names are also acceptable (e.g. Smi Jo / Smi) and it is case insensitive (e.g. smiT / Smit ). For wildcard searching, "%" can be added to the name as required (e.g. Smit% Joh%, %vid %lay, %poulo%). If no "%" is added it will be considered as %name%.

During foucus out, if it has a single result, it will be auto-populated else the popup with the search results will be shown. Select the name from the list. If in the Site Contact or Project Coordinator they do not need to be listed here.
23 Comments: Additional details
24 Upload file: Attach quotes, SOW, Change Orders and any other documents supporting this request.

When the ERFE has been submitted for approvals, the document cannot be altered. Contact TTS Purchasing and Asset Team (Dawn Gustafson 733-9035 or Jodi Hammarsten 733-6699) if you need to make adjustments.
